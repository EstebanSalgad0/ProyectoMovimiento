# App Flutter - Proyecto Movimiento

Esta carpeta contiene la aplicación móvil en Flutter del prototipo Proyecto Movimiento. La app funciona como la interfaz visual principal para conectarse con el servidor de análisis y mostrar la retroalimentación generada a partir del video.

La documentación completa del proyecto, la explicación general y la guía principal de ejecución están en el README raíz del repositorio.

## Uso rápido

1. Abre una terminal en la carpeta `app`.
2. Instala dependencias con `flutter pub get`.
3. Ejecuta la app con `flutter run`.

## Requisitos

- Flutter SDK instalado.
- Emulador Android o dispositivo físico configurado para depuración.

## Nota

Este módulo forma parte de un prototipo de análisis de movimiento. Para detalles de conexión con el servidor, flujo de prueba y contexto del proyecto, revisa el README principal.
